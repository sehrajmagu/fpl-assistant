# FPL Assistant

A Python-based tool designed to help Fantasy Premier League (FPL) players make more informed decisions using player and team performance data.

## Features

### Top Performing Players
Displays the highest performing players in the league for the current season based on key metrics such as total points, goals, and assists. This helps identify players who are consistently delivering strong returns.

### Team-wise Statistics
Provides an overview of team performance across the league. Users can explore and compare different teams to better understand strengths, weaknesses, and overall form.

### Player Comparison
Allows users to compare two players side by side. Key statistics such as total points, goals, assists, and overall performance are presented to support better transfer and selection decisions.

### Captain Picker
Takes a user’s FPL team as input and suggests the best captain choice. The recommendation is based on player performance and consistency, aiming to maximize potential points for the gameweek.

## Tech Stack

- Python
- Data handling and analysis libraries (e.g., pandas, numpy)
- FPL API
